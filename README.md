# 💼 Alok Landing Page

## 📌 Project Title
Alok Landing Page – Responsive Modern Website

---

## 📌 Project Overview
Alok Landing Page is a fully responsive single-page website developed using HTML, CSS, and JavaScript.

This project features an interactive fixed navigation menu that dynamically changes style on scroll and hover, designed with a modern UI to deliver a smooth and engaging user experience across all devices.

---

## 🚀 Features

📌 Fixed responsive navigation bar

🎨 Scroll-based navbar background effect

🖱️ Hover underline animation on menu items

📱 Fully responsive design (mobile + desktop)

⚡ Smooth scrolling between sections

💎 Modern hero section with CTA button

🧩 Interactive feature cards with hover effects

🔘 Mobile hamburger menu

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome Icons
